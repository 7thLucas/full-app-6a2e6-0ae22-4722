Create a modern, premium, and highly interactive E-commerce Marketing App and Website inspired by Flipkart, Amazon, and Meesho. The UI should look extremely clean, stylish, and user-friendly.

Key UI/UX & Design Requirements:
1. Top Bar: Clean header with a prominent search bar (with voice search icon), a shopping cart icon, and a user profile button.
2. Hero Section: An eye-catching sliding banner carousel showcasing the latest deals, discounts (e.g., "50% OFF"), and trending marketing campaigns.
3. Category Grid: A horizontal scrolling section with modern rounded icons for product categories (e.g., Fashion, Electronics, Beauty, Home Decor).
4. Product Showcase: A grid layout displaying high-quality product cards. Each card must include:
   - High-resolution product image.
   - Product title and short description.
   - Price, original price strikethrough, and a bold discount percentage tag.
   - Customer rating badge (e.g., "4.5 ★").
   - A smooth "Add to Cart" button.
5. Bottom Navigation Bar: Sleek and sticky navigation bar with icons for "Home", "Categories", "My Cart", "Orders", and "Profile".
6. Theme & Feel: Vibrant yet professional color palette (like Amazon/Meesho). It must support a dark mode toggle for a premium look. Ensure smooth animations and fast transitions between screens.
