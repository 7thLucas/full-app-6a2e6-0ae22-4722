The Metric Vibecart Lives By: Cart Additions / Week

A Verified Operation is a shopper tapping "Add to Cart" in the deployed app — a real, completed, domain-meaningful action that directly precedes revenue. Not a page view, not a login.

Weekly Verified Operations (WVO) — Formula

Assumption — adjust: Weekly active shoppers = A
Assumption — adjust: Sessions per shopper per week = S
Assumption — adjust: Avg cart additions per session = C

A shoppers × S sessions/wk × C adds/session

= A × S × C cart additions / week

Example: 500 shoppers × 2 sessions × 3 adds = 3,000 WVO / week

Projected Value per Week

Assumption — adjust: Purchase conversion rate = R%
Assumption — adjust: Average order value = ₹V

WVO × R% × ₹V = weekly revenue facilitated

Weekly: WVO × R% × ₹V

Annual: × 52 weeks

Example: 3,000 × 5% × ₹1,200 = ₹1,80,000 / week → ₹93.6L / year
